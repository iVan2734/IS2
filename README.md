# Game Theory Algorithms

This project aims to implement various algorithms and concepts from game theory. The goal is to provide a clear understanding of how these algorithms work and their applications in real-world scenarios. 

## Contents
- **Introduction**: Overview of game theory and its importance in decision-making.
- **Algorithms**: Detailed descriptions of the algorithms implemented in this project, including but not limited to:
  - Minimax Algorithm
  - Nash Equilibrium
  - Dominance Solvable Games
- **Usage**: Instructions on how to use the implemented algorithms.
- **Examples**: Practical examples showcasing the application of the algorithms in different situations.
- **Contributions**: Guidelines for contributing to the project.

## Getting Started
To get started with this project, clone the repository and navigate to the project directory:
```bash
git clone https://github.com/iVan2734/IS2.git
cd IS2
```
